# Complete CI CD pipeline for a go web application

### Run & Test application locally
- Clone repo
```
git clone https://github.com/anilabhabaral/ci-cd-go.git
```
- Build application
```
cd ci-cd-go
go build .
```
- Run the Tests
```
go test -v 
```
- Run application
```
./ci-cd-go
```
- Access application
```
curl localhost:8080/greet
curl localhost:8080/hello/Anilabha
```
